# Connect Four

Реализация классической игры Connect Four на языке Python.
Проект предназначен для демонстрации базовых алгоритмов и работы с двумерными структурами данных.

---

## Описание

Программа представляет собой консольное приложение для двух игроков.
Игроки поочередно выбирают колонку, в которую помещается фишка.

Победа фиксируется при формировании последовательности из четырёх одинаковых элементов:

* по горизонтали
* по вертикали
* по диагонали

---

## Функциональность

* Инициализация игрового поля фиксированного размера
* Обработка пользовательского ввода
* Моделирование “падения” фишки в колонке
* Проверка условий победы
* Смена активного игрока

---

## Структура

Ключевые компоненты реализации:

* представление игрового поля в виде двумерного массива
* основной игровой цикл
* функции проверки победных комбинаций
* обработка граничных условий
